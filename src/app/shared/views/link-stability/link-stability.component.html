About Time Deviation (TDEV)

Time Deviation (TDEV) is a measure of frequency stability that characterizes the time error accumulated over different averaging intervals. This analysis is calculated using RefSys Diff data for location {{ dataIdentifier | titlecase }}, derived from the Modified Allan Deviation (MDEV).

Formula: TDEV(τ) = τ × MDEV(τ) / √3
